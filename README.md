# Toddzilla Ghost Theme

Ghost blog running at https://toddbirchard.com

This repository contains the entire blog instance, including the custom theme. 

## Features
- Multiple post types
- Related articles widget
- Experience widget
- Twitter widget
- Gitub Widget

## Stack
- NodeJS
- Ghost
- Express
- Handblebars
- Sass
- Gulp
- Grunt (for Ghost)
